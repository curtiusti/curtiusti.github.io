# My Portfolio

A personal portfolio site built with plain HTML/CSS/JS, hosted on GitHub Pages.

## Structure

- `index.html` — main page (about, skills, projects, contact)
- `assets/css/style.css` — styles, light/dark theme support
- `assets/js/script.js` — theme toggle + footer year
- `_config.yaml` — GitHub Pages site config

## Customizing

1. Replace the placeholder text in `index.html` (name, tagline, about, projects, links).
2. Swap the avatar `<img>` `src` for your own photo.
3. Update the badges — tech stack badges and the contact links — to match you.
4. More badges: https://shields.io

## Local preview

Just open `index.html` in a browser, or serve the folder with any static file server.
